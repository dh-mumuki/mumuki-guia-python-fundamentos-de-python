Una variable es un espacio de memoria que podemos utilizar para almacenar valores.<br>
Si queremos utilizar una variable lo primero que debemos hacer es asignarlo un valor.<br>
¿Cómo hacemos eso?<br>

``` python
nombre_de_la_variable = valor
```
